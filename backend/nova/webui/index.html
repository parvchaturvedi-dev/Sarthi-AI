One-time setup (OAuth app):
1. Google Cloud Console → new project
2. Enable Gmail, Calendar, Drive & People APIs · 3. OAuth consent (External) → add your email as a test user
4. Credentials → OAuth client ID → Desktop app → download JSON
5. Save it in the project folder as nova_gmail_credentials.json, then click "Add connection".
